# *kevao-css*
## Repositorio para css predefinido
---
- Padding
  * Left: px, %, rem.
  * Right: px, %, rem.
  * Top: px, %, rem.
  * Bottom: px, %, rem.
---
- Margin
  * Left: px, %, rem.
  * Right: px, %, rem.
  * Top: px, %, rem.
  * Bottom: px, %, rem.
---
- Width: px, %, vw.
---
- Height: px, %, vh.
---
## Keywords
- vh: Viewport Height.
- vw: Viewport Width.
- rem: relative unit that'll always be based upon the font-size value of the root element.
- px: Pixels.
- %: Percentage.
